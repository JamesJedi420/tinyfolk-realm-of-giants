# Tinyfolk asset and systems reference board

## Purpose

Track reusable references for Tinyfolk: Realm of Giants across environment art, systems, UI, audio, animation, and Roblox-native implementation patterns.

## Decision rules

- Prefer stylized, readable, reusable assets
- Prefer deterministic, inspectable systems over opaque black-box kits
- Prefer references compatible with persistent-world and validator-driven design
- Avoid unclear licensing, brittle kits, and strong style mismatch

Mark every item as:

- ship candidate
- prototype-only
- reference-only
- avoid
